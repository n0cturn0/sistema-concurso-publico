# sistema-concurso-publico
Sistema de Inscrição para concurso público usado pelo sistema colégio militar de ensino, no colégio militar de Campo Grande. PHP / Codeigniter / Ajax / Js / Bootstrap / PostgreSql Usabilidade: Registra resposáveis e candidatos, emite boleto para pagamento.Processa arquivo de retorno bancário em lote para processamento ou não da inscrição. Rotina de pagamento, faz a liberação ou não para a impresssão do documento que contém informações pertinentes ao candidatos, informações gerais de execução do certame.
